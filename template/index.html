How to use?

	Importants:
	The search filtering is based on Filterizr JS: https://github.com/giotiskl/filterizr
	The code highlighter is based on Prism JS: https://github.com/PrismJS/prism

	Basic Usage:
	[HTML/Markup] 
	change <pre/> class to language-markdown
	Paste your code in to <code><!-- <p>Hello world</p> --></code> wrap all contents with HTML comment
	
	[Javacript] 
	change <pre/> class to language-js
	Paste your code in to <code>function(){...}</code>
	
	[CSS] 
	change <pre/> class to language-css
	Paste your code in to <code>body{background:black}</code>
	
	[PHP] 
	change <pre/> class to language-php
	Paste your code in to <code>echo "Hello world"</code>
